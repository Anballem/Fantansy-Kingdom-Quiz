<h1>Fantasy Kingdom Quiz </h1>

#Project Overview
This project is a single-page, interactive personality quiz built using HTML, CSS, and JavaScript.
Users answer a series of multiple-choice questions and receive a result at the end based on their selections.


The quiz theme is “Which Fantasy Kingdom Do You Belong To?”, where answers are automatically scored and the final result is calculated without requiring the user to count their responses.

#Features
. Single-page application (no page reloads)
. At least 5 multiple-choice questions
. Automatic scoring system
. Questions and results are hidden and revealed dynamically
. Intro information displayed only at the start
. Styled using Tailwind CSS
. Simple, responsive layout
. JavaScript functions reused where possible


#Technologies Used
. HTML5
. CSS3
. JavaScript (Vanilla JS)
. Tailwind CSS (via CDN)


#How the Quiz Works
1. The user is presented with an introduction screen.
2. When the quiz starts, one question is shown at a time.
3. Each answer contributes to a hidden score category.
4. After all questions are answered, the quiz calculates the highest score.
5. A personality result is displayed based on the final score.
